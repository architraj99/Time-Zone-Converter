# Time Zone Converter

This is a simple time zone converter made using HTML, CSS and JavaScript.

You can select a country, enter a date and time, and convert it to another country's time. It helps in quickly checking time differences without doing manual calculations.

---

-----> Features

- convert time between different countries  
- supports multiple regions (India, USA, Europe, Asia, etc.)  
- use current time button  
- swap time zones  
- reset option  
- shows detailed result (from, to, original, converted)  
- simple and clean UI  

---

----> How to use

1. select "from" time zone  
2. select date and time  
3. select "to" time zone  
4. click convert  

You can also:
- use current time button  
- swap time zones  
- reset everything
